# SupplierMainDocument

#### Field Names:

* id - String
* type - String
* current - SupplierSubDocument
* historical - \[SupplierSubDocument!]

#### Example:

```
{
  "id": "abc123",
  "type": "abc123",
  "current": SupplierSubDocument,
  "historical": [SupplierSubDocument]
}
```
