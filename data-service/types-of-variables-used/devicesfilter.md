# DevicesFilter

#### Field Names:

* `supplier - String`
* `owner - String`
* `visible - Boolean`
* `active - Boolean`
* `connected - Boolean`
* `whitelisted - Boolean`

#### Example:

```
{
  "supplier": "xyz789",
  "owner": "abc123",
  "visible": false,
  "active": true,
  "connected": false,
  "whitelisted": true
}
```
