# DeviceMainDocument

#### Field Names:

* `id - String`
* `type - String`
* `current - DeviceSubDocument`
* `historical - [DeviceSubDocument!]`

#### Example:

```
{
  "id": "abc123",
  "type": "abc123",
  "current": DeviceSubDocument,
  "historical": [DeviceSubDocument]
}
```
