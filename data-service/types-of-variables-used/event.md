# Event

#### Field Names:

* `id - ID!`
* `txHash - String!`
* `sender - String`
* `device - String`
* `assetId - String`
* `action - String`
* `status - String`
* `createdAt - DateTime`
* `updatedAt - DateTime`
* `score - Float`

#### Example:

```
{
  "id": ID,
  "txHash": "xyz789",
  "sender": "abc123",
  "device": "abc123",
  "assetId": "xyz789",
  "action": "xyz789",
  "status": "abc123",
  "createdAt": DateTime,
  "updatedAt": DateTime,
  "score": 987.65
}
```
