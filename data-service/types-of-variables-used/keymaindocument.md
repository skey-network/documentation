# KeyMainDocument

#### Field Names:

* `id - String`
* `type - String`
* `current - KeySubDocument`
* `historical - [KeySubDocument!]`

#### Example:

```
{
  "id": "xyz789",
  "type": "xyz789",
  "current": KeySubDocument,
  "historical": [KeySubDocument]
}
```
